# Security policy

## Supported versions

| Version    | Supported |
| ---------- | --------- |
| latest 0.x | Yes       |
| older 0.x  | No        |

Only the latest release receives security patches. Upgrade to stay covered.

## Reporting a vulnerability

**Do not open a public issue for security vulnerabilities.**

Use [GitHub Security Advisories](https://github.com/itsoltech/canopy-desktop/security/advisories/new) to report privately. This keeps the details confidential until a fix is available.

When reporting, include:

- Description of the vulnerability
- Steps to reproduce
- Affected version(s)
- Impact assessment if possible

## What to expect

- Acknowledgment within 48 hours
- Status update within 7 days
- If accepted: fix released as a patch, credited in the changelog (unless you prefer anonymity)
- If declined: explanation of why
